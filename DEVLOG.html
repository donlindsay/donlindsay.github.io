Net result

The stack runs on two pillars — AllegroGraph (symbolic) + Ollama (neural) — with a diffusers SD worker and a local FastAPI/Gradio frontend. The web tier is an 8-peer cloud council synthesized locally. n8n, ComfyUI, and Docker are all gone from the hot path (n8n and ComfyUI survive only as one-click swap-in pedals). Everything is graph-native: data, memory, provenance, custody, the DCAT registry, and now the credential inventory all live in AllegroGraph.
