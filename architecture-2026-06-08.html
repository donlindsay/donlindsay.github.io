AI Stack — current architecture (2026-06-08)

id: urn:kg:doc:architecture-2026-06-08 · source: audit · date: 2026-06-08

AI Stack current architecture (2026-06-08). The local LLM runs on a free engine x model matrix called Reaper. Engines: Ollama and llama.cpp. Models: Nemotron and Twin-Llama. The PRIMARY engine is llama.cpp (llama-server, OpenAI-compatible API on port 8002, built for CUDA sm_120 / Blackwell), and the default engaged quadrant is llama.cpp x Nemotron (nemotron-3-nano:4b). vLLM was the prior GPU serving engine and is now SHELVED (dormant): out of the active matrix, ai-vllm stopped, but still swap-in-able via set_engine('vllm'). AllegroGraph 9.0.1 is the knowledge graph + RAG store; Ollama also serves embeddings (nomic-embed-text). The 12 GB GPU (RTX 5070) is single-resident: engaging one GPU engine evicts the others. Stable Diffusion runs as a socket-activated diffusers worker.
