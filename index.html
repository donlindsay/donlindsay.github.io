MX-RDF is an (RDF) framework for Emacs (MX)

W3C grammars in elisp. MX-RDF features these grammars as the core of a RDF compatible application for Emacs Org Mode (org-mode). Org Mode provides a convenient API for describing various RDF based grammars in both elisp and org files. Functions that operate on the grammars can apply to a wide range of other applications and data management tasks, with a powerful array of endpoints for data storage repositories worldwide.

Resource Description Framework

Within the W3C corpus of Recommendations and Specifications are several grammars for operating on Linked Data. Linked Data is a set of initiatives promoting the facilitation of expressive data management and access to remote data repositories located on both private servers and also publically available. Representations of these grammars are possible in elisp and several examples are provided, along with documentation and example code.
