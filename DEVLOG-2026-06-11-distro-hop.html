DEVLOG 2026-06-11 — Distro Hop

Drafted by SHELL (aistack cli agent); persisted + one fact corrected by the operator.

The AI stack was consolidated from a 5-distro sprawl into a single WSL2 distro.

  • One distro now. Migrated from the old Ubuntu WSL2 distro (now deleted, ~124 GB reclaimed) into Ubuntu-24.04, which hosts the whole stack.
  • Universal model store at /srv/models (live GGUFs + SD + whisper, ~6.8 GB); ~33 GB of stale HF/ollama duplicates dropped.
  • llama.cpp rebuilt native with the latest CUDA 13.3 + llama.cpp master (Blackwell sm_120; BLACKWELL_NATIVE_FP4 enabled), serving on-box nemotron-3-nano-4b on :8002.
  • AllegroGraph 9.0.1 migrated via agtool archive backup-all/restore-all (online, zero downtime). Repos trimmed 21 → 17; lean-rebuilt 8.5 GB → ~970 MB by setting TransactionLogSize 8m + DesiredTlogFiles 1 (AG pre-allocates two 256 MB logs per repo by default). Triple counts verified identical (0 mismatches).
  • aistack migrated; main venv rebuilt from pip freeze (92 pkgs); all model paths rewired to /srv/models.
  • Clean cutover. Stopped U1, ran start-stack.sh in U2 + ai-llamacpp; all services green — AG :10035, llama.cpp :8002, API :8765 (every component true), embed :11434, console :7860, SD :8189. Nemotron tight-loop verified.
  • NemoClaw runs alongside in Docker Desktop, unaffected (routes to cloud NIM).

Result: the tight local core — AllegroGraph + llama.cpp + on-box Nemotron + aistack — now runs natively in one distro on the RTX 5070, off a single model store.
